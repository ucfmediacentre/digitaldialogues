Server Side Modules

Flash RTMP Player

Installation

In the /examples/ServerSideModules directory:

  • LINUX
    Run ./installall.sh
  • WINDOWS 7 / WINDOWS 8
    Right-click installall.bat and then select Run as administrator
  • WINDOWS SERVER
    Double-click the installall.bat file
  • MAC OS X
    Double-click the installall.command file

To Build

Follow the instructions in the Wowza IDE: User's Guide.
